### 1.1 Schema

HugeGraph 提供单一接口获取某个图的全部 Schema 信息，包括：PropertyKey、VertexLabel、EdgeLabel 和 IndexLabel。

##### Method & Url

```
GET http://localhost:8080/graphs/hugegraph/schema
```

##### Response Status

```json
200
```
